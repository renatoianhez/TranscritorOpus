**Transcritor Opus**

Esse é um programinha para a transcrição de arquivos de áudio gerados pelo WhatsApp (.opus).

![2026-01-15_105753](https://github.com/user-attachments/assets/18d93960-bb47-4116-84f7-769bbba2a6ee)

Na primeira execução ele fará o download do modelo escolhido. Se usou o instalador (em Releases), será necessário executar em modo Administrador na primeira vez, porque ele salva este arquivo de modelo na pasta do programa.
